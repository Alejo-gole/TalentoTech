Introducción al Proyecto

En esta sección se explica la lógica general del frontend.
El sitio utiliza una estructura HTML modular, con divisiones principales para cada parte del contenido. Cada bloque está encapsulado en un section o div que permite mantener el código ordenado y fácilmente manipulable.
Las clases de Tailwind como bg-bg-section, text-text-dark o rounded-md usan variables CSS personalizadas (definidas mediante --color-*), lo que facilita el cambio de temas dinámicos sin reescribir estilos.

Lógica de Cambio de Tema con JavaScript

El botón “Cambiar modo” ejecuta un script que altera las variables CSS globales del documento (:root) para actualizar instantáneamente la apariencia del sitio. El código utiliza una estructura switch para rotar entre tres modos:

  • Modo Claro (predeterminado): paleta neutra y luminosa.
  • Modo Oscuro: tonos fríos y alto contraste.
  • Modo Kawaii: colores cálidos, alegres y contrastantes.

Uso de Tailwind CSS y Variables de Tema

Esta parte explica cómo se configuran y aplican los estilos del sitio. El proyecto utiliza Tailwind CSS compilado mediante el comando:
npx @tailwindcss/cli -i css/input.css -o css/main.css --watch
De esta forma, cada cambio en el archivo input.css se refleja automáticamente en el archivo main.css, manteniendo un flujo de trabajo en vivo durante la edición.
Las variables CSS definidas con --color-* permiten que Tailwind use los mismos nombres de color en todas las clases. Esto garantiza coherencia visual y facilita la integración con JavaScript para cambiar los temas en tiempo real.

Composición Visual y Efectos Interactivos

La galería muestra cómo aplicar efectos visuales y transiciones suaves con Tailwind.
Cada imagen está dentro de un contenedor con clases como grayscale hover:filter-none y hover:scale-105, lo que crea un efecto de enfoque y zoom al pasar el cursor.
Estas animaciones están optimizadas con transition-transform duration-200, que asegura una respuesta fluida sin afectar el rendimiento.
Además, las imágenes incluyen object-cover y rounded-md para mantener proporciones equilibradas y esquinas suaves.
